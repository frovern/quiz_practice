# QuizRoom — веб-приложение для интерактивных квизов

## Функциональность:
- Регистрация и авторизация (организатор / участник)
- Создание квизов (настройка времени, категории)
- Добавление вопросов с текстом, изображением, одиночным/множественным выбором
- Запуск квиза с генерацией уникального кода комнаты
- Подключение участников по коду
- Синхронный показ вопросов с таймером 
- Подсчёт баллов 
- Лидерборд в конце квиза
- Личный кабинет с историей


## Технологии
Frontend - React 
Backend - Node.js
WebSocket - Socket.IO
База данных - MySQL (XAMPP) 
 
## Установка и запуск

### Требования
- Установленный [XAMPP](https://www.apachefriends.org/)
- Установленный [Node.js](https://nodejs.org/) 

### Запуск базы данных
1. Открыть XAMPP Control Panel
2. Запустить **Apache** и **MySQL**
3. Перейдите в phpMyAdmin: `http://localhost/phpmyadmin`
4. Создайте базу данных с именем `quiz`
5. Импортируйте файл `database/quiz.sql`

### Запуск сервера (backend)
```bash
cd backend
npm install
npm run dev

### Запуск клиента (frontend)
```bash
cd frontend
npm install
npm start
